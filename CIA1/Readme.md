# Event Booking Website

## How to Run

1. Clone the repository

```bash
git clone https://github.com/manan-hemani/FSD-Lab.git
```

2. Navigate to the Folder

```bash
cd CIA1
```

3. Open 'index.html' in your browser
4. Navigate using the top navigation bar and trying booking an event.
